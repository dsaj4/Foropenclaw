## 二、固定框架详解（强制继承）

### 2.1 执行摘要

**功能定位**：报告精华浓缩，供快速阅读

**标准结构**：
| 子章节 | 内容要求 | 建议篇幅 |
|--------|---------|---------|
| 核心投资逻辑 | 用3-5句话概括投资核心理由 | 200-300字 |
| 关键财务指标速览 | 营收、利润、增速、估值核心数据 | 表格形式 |
| 主要风险提示 | 列出3-5个最核心风险点 |  bullet points |

**写作规范**：
- 必须独立成页
- 数据必须与正文一致
- 结论必须可在正文找到支撑

---

### 2.2 第一章：公司概况

**功能定位**：建立对公司的基础认知

**标准结构**：
```
1.1 基本信息
    - 公司全称、证券代码、上市地点
    - 主营业务一句话概括
    - 最新市值与股价（截至报告日期）

1.2 发展历程
    - 里程碑事件时间线
    - 发展阶段划分（初创/成长/成熟）

1.3 股权结构
    - 实际控制人/控股股东
    - 前十大股东（如有重大变化需说明）

1.4 管理团队
    - 核心高管背景
    - 团队稳定性评估
```

**继承规则**：
- 章节标题**原样保留**
- 仅需替换主体名称和数据
- 分析维度保持不变

---

### 2.3 第二章：行业分析

**功能定位**：理解公司所处外部环境

**标准结构**：
```
2.1 行业规模与增速
    - 当前市场规模（TAM/SAM/SOM）
    - 历史增速与未来预测
    - 行业生命周期判断

2.2 竞争格局
    - 市场份额分布（CR5/CR10）
    - 主要竞争对手画像
    - 公司市场地位

2.3 行业趋势
    - 技术趋势
    - 需求趋势
    - 商业模式演进

2.4 政策环境
    - 行业监管框架
    - 近期政策变化
    - 政策影响评估
```

**继承规则**：
- 行业分析维度固定
- 具体数据和判断根据目标公司所在行业调整

---

### 2.4 第三章：业务分析

**功能定位**：深入理解公司商业模式和竞争力

**标准结构**：
```
3.1 主营业务构成
    - 收入结构拆分（按产品/地区/渠道）
    - 各业务板块增速对比
    - 业务结构变化趋势

3.2 产品/服务体系
    - 核心产品介绍
    - 产品生命周期分析
    - 新产品管线（如适用）

3.3 商业模式
    - 价值主张
    - 盈利模式
    - 关键资源与能力

3.4 核心竞争力
    - 竞争优势来源（成本/差异化/聚焦）
    - 护城河评估（品牌/技术/网络效应/转换成本）
    - 可持续性分析
```

**继承规则**：
- 业务分析框架通用
- 竞争力评估维度根据行业特性微调

---

### 2.5 第四章：财务分析

**功能定位**：通过财务数据验证业务表现

**标准结构**：
```
4.1 盈利能力分析
    - 收入增长分析（总量+增速）
    - 利润率分析（毛利率/净利率/ROE）
    - 盈利质量评估

4.2 运营效率分析
    - 资产周转率
    - 存货/应收账款周转
    - 费用率分析

4.3 偿债能力分析
    - 资产负债率
    - 流动比率/速动比率
    - 利息覆盖倍数

4.4 现金流分析
    - 经营现金流
    - 自由现金流
    - 现金流与利润匹配度

4.5 同业对比
    - 关键指标同业排名
    - 相对优势/劣势识别
```

**继承规则**：
- 财务分析维度标准化
- 对比公司根据目标公司所在行业调整

---

### 2.6 第六章：盈利预测与估值

**功能定位**：提供投资决策的量化依据

**标准结构**：
```
6.1 关键假设
    - 收入假设（量价拆分）
    - 成本假设
    - 费用假设

6.2 盈利预测
    - 未来3年收入预测
    - 利润预测
    - 敏感性分析

6.3 估值分析
    - 绝对估值（DCF）
    - 相对估值（PE/PB/PS同业对比）
    - 估值结论与目标价
```

**继承规则**：
- 预测框架固定
- 估值方法根据公司类型选择（成长股用PS，价值股用PE/PEG）

---

### 2.7 第七章：风险提示

**功能定位**：揭示投资的不确定性和潜在损失

**标准结构**：
```
7.1 经营风险
    - 业务相关风险（需求波动/竞争加剧/技术替代）
    - 供应链风险
    - 人才风险

7.2 财务风险
    - 现金流风险
    - 债务风险
    - 汇率/利率风险

7.3 市场风险
    - 宏观经济风险
    - 政策监管风险
    - 资本市场波动风险
```

**继承规则**：
- 风险分类框架固定
- 具体风险点根据目标公司特性定制

---

### 2.8 第八章：投资要点

**功能定位**：总结投资建议

**标准结构**：
```
8.1 投资亮点
    - 核心增长驱动因素
    - 竞争优势
    - 催化剂

8.2 关注要点
    - 需要持续跟踪的指标
    - 潜在风险信号

8.3 评级与目标价
    - 投资评级（买入/增持/中性/减持）
    - 目标价及推导逻辑
    - 投资期限建议
```

**继承规则**：
- 投资要点结构固定
- 结论必须与正文分析一致

### 3.3 特殊框架适配规则

| 判断维度 | 保留条件 | 删除条件 | 替换规则 |
|---------|---------|---------|---------|
| 行业匹配 | 目标公司属于框架适用行业 | 目标公司不属于框架适用行业 | - |
| 业务特征 | 目标公司具有框架描述的特征 | 目标公司不具备该特征 | 替换为对应特征的框架 |
| 数据支撑 | 有公开数据支撑该框架分析 | 缺乏必要数据 | 简化为概述段落 |
| 重要性 | 该维度对投资决策影响显著 | 影响可忽略 | 降级为子章节 |

## 四、框架使用指南

### 4.1 新报告撰写流程

```
步骤1：确定目标公司类型
        │
        ▼
步骤2：加载固定框架（100%继承）
        │
        ▼
步骤3：从特殊框架库筛选适用框架
        │
        ▼
步骤4：根据目标公司特性调整特殊框架
        │
        ▼
步骤5：生成新报告三级目录
        │
        ▼
步骤6：标注每章节的继承来源
```

```
新标杆报告分析
        │
        ▼
┌─────────────────────┐
│ 发现新的有效框架？   │

        │

    │      │
    ▼      ▼

│ 评估通用性   │  │ 发现框架优化点？     │
│ 和可复用性   │  └─────────────────────┘
└─────────────┘          │
        │            是 ─┴─ 否
    高 ─┴─ 低           │      │
    │      │            ▼      ▼
    ▼      ▼      ┌─────────┐  ┌─────┐
加入特殊框架库   作为单次参考  │ 更新参数 │  保持现状
（版本+0.1）   （不加入库）  └─────────┘  └─────┘
                              （版本+0.0.1）
```
